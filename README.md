## **개발 환경**

사용 언어 : C++, Object-C

게임 엔진 : cocos-2d

물리엔진 : box2d

platform : IOS

## **소개 영상**

## **구현**

-   저장 기능
-   Stage 잠금/해제 기능
-   Stage 단위 미션 기능
-   Life/HP 기능
-   코인 / Score 획득 기능
-   클리어 타임 기능
-   게임 패드 (좌/우, 점프) 기능
-   상호 작용 가능한 물리 오브젝트 추가
-   부실수 있는 상자 ( 부실경우 아이템 획득)
-   몬스터 / 몬스터 AI 추가
-   씬전환을 위한 포탈 기능
